# Openly

Openly is an independent agent home insurance platform founded in 2017 and headquartered in Boston, Massachusetts. The company operates as a general agency and program administrator, offering homeowners insurance policies exclusively through independent insurance agents. Openly provides REST APIs enabling tech-savvy and larger independent agencies to build custom interfaces and plug into the Openly platform on the backend for quoting, binding policies, managing endorsements, and processing renewals.

## API

Openly's API is a partner-only integration available to appointed independent insurance agencies. The developer portal is hosted at [developers.openly.com](https://developers.openly.com/) using Stoplight. Authentication is required via GitHub, Okta, or email credentials. Specific API documentation, rate limits, and authentication credentials are provided to approved agency partners as part of the agency onboarding process.

### Capabilities

- Generate homeowners insurance quotes (primary, secondary/seasonal, and landlord policies)
- Bind policies programmatically
- Manage endorsements (water backup, equipment breakdown, home-sharing, personal cyber, etc.)
- Process policy renewals
- Access policy documents, billing, and reports

### Coverage

- Up to $5M guaranteed replacement cost
- Up to $1M liability
- Available in 24 U.S. states with nationwide expansion planned
- Underwritten by MS Transverse and Clear Blue Insurance Group
